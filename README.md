# 检验科查询工具

一个功能完善的检验科信息查询系统，提供多种查询模块，帮助检验科工作人员快速获取所需信息。

## 功能模块

### 1. 项目性能查询
- 按项目名称、编码、关键词搜索
- 按分类和类型筛选
- 显示性能指标、参考范围、灵敏度、特异性等详细信息

### 2. 试剂存放查询
- 按试剂名称、货号、规格搜索
- 按存放位置和状态筛选
- 实时显示试剂状态（正常、临期、过期）
- 支持库存管理信息查看

### 3. 试剂说明书查询
- 按试剂名称、品牌、型号搜索
- 在线预览和下载功能
- 版本管理和说明文档管理

### 4. 文件检索
- 按文件名称、类型、关键词搜索
- 支持SOP、质控、培训、其他文件类型
- 文件版本管理和作者信息
- 在线预览和下载功能

## 技术特点

- **纯前端实现**：HTML + CSS + JavaScript，无需后端
- **响应式设计**：适配桌面和移动设备
- **现代化UI**：采用渐变背景、卡片式布局
- **易于使用**：直观的标签页导航和搜索功能
- **数据分离**：数据存储在独立的data.js文件中，便于维护和更新

## 使用方法

### 1. 打开应用
直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 数据录入
打开 `data.js` 文件，按照示例数据格式填入您的实际数据：

#### 项目性能数据格式
```javascript
{
    id: 1,
    name: "项目名称",
    code: "项目编码",
    category: "分类",
    type: "类型",
    performance: "性能指标",
    reference: "参考范围",
    method: "检测方法",
    sensitivity: "灵敏度",
    specificity: "特异性",
    lastUpdate: "2026-03-13"
}
```

#### 试剂存放数据格式
```javascript
{
    id: 1,
    name: "试剂名称",
    catalogNumber: "货号",
    specification: "规格",
    manufacturer: "生产厂家",
    location: "存放位置",
    temperature: "保存温度",
    quantity: "库存数量",
    batchNumber: "批号",
    expiryDate: "有效期",
    status: "状态",
    lastUpdate: "2026-03-13"
}
```

#### 试剂说明书数据格式
```javascript
{
    id: 1,
    name: "试剂名称",
    brand: "品牌",
    model: "型号",
    catalogNumber: "货号",
    version: "版本号",
    uploadDate: "上传日期",
    fileFormat: "文件格式",
    description: "说明",
    filePath: "文件路径"
}
```

#### 文件检索数据格式
```javascript
{
    id: 1,
    name: "文件名称",
    type: "文件类型",
    category: "分类",
    version: "版本号",
    author: "作者",
    uploadDate: "上传日期",
    fileSize: "文件大小",
    format: "格式",
    keywords: "关键词",
    description: "说明",
    filePath: "文件路径"
}
```

### 3. 查询操作
1. 点击顶部标签页切换到对应的查询模块
2. 在搜索框中输入关键词
3. 可选择筛选条件进行精确查找
4. 点击"搜索"按钮或按回车键执行查询
5. 查看搜索结果，可点击"查看详情"、"在线预览"或"下载"按钮

## 自定义配置

### 修改主题颜色
在 `styles.css` 文件中修改 `:root` 部分的CSS变量：

```css
:root {
    --primary-color: #007bff;      /* 主色调 */
    --success-color: #28a745;      /* 成功色 */
    --warning-color: #ffc107;      /* 警告色 */
    --danger-color: #dc3545;       /* 危险色 */
    /* ... 其他颜色变量 */
}
```

### 添加新的筛选条件
在对应的HTML文件中添加 `<select>` 元素，并在 `script.js` 的搜索函数中添加相应的筛选逻辑。
